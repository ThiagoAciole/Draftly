# SVG Icon Library

Arquivos SVG gerados para mapeamento visual e importacao no Figma.

- `ui/`: icones extraidos do catalogo usado pelo componente `SvgIcon`.
- `markdown-actions/`: icones usados pela toolbar e pelo slash menu.
- `markdown-html/`: icones inseridos como HTML no preview Markdown.
- `masks/`: icones usados como CSS masks.

Total: 109 arquivos SVG.

O app continua usando os componentes atuais; esta pasta existe como biblioteca de referencia/exportacao.
